PROJET 02

Virtualisation et Conteneurisation

Durée : 3 séances
Équipe : 5 personnes
Réalisé : Avril 2026

Objectifs du projet

  • Maîtriser les environnements virtualisés avec Proxmox/VMware
  • Découvrir Docker et la conteneurisation d'applications
  • Apprendre l'orchestration avec Docker Compose
  • Initiation à Kubernetes (mini-cluster)

Architecture mise en place

[Schéma de virtualisation à insérer]

Proxmox (hyperviseur) → 3 VMs (Ubuntu) → Docker sur chaque VM

Commandes importantes

Installation de Proxmox

# Télécharger l'ISO de Proxmox VE
# Installer sur un serveur physique

# Après installation, accès web : https://IP:8006

# Créer des templates de VM
pveam update
pveam available
pveam download local ubuntu-22.04-standard_22.04-1_amd64.tar.zst

Docker

# Installation de Docker
sudo apt update
sudo apt install docker.io -y
sudo systemctl start docker
sudo systemctl enable docker

# Vérification
sudo docker --version
sudo docker run hello-world

# Commandes de base
sudo docker images
sudo docker ps
sudo docker ps -a
sudo docker stop [CONTAINER_ID]
sudo docker rm [CONTAINER_ID]

Docker Compose

# Installation de Docker Compose
sudo apt install docker-compose -y

# Exemple de fichier docker-compose.yml
version: '3'
services:
  web:
    image: nginx:latest
    ports:
      - "80:80"
  db:
    image: mysql:5.7
    environment:
      MYSQL_ROOT_PASSWORD: example

# Lancer les services
sudo docker-compose up -d
