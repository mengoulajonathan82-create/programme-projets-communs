PROJET 04

Sécurisation : Firewall & IDS/IPS

Durée : 3 séances
Équipe : 5 personnes
Réalisé : Mai 2026
Niveau : Intermédiaire

Objectifs du projet

  • Configurer un firewall avancé (Pfsense/OPNsense) pour protéger le réseau
  • Mettre en place des règles de filtrage et du NAT
  • Installer et configurer un VPN (WireGuard/OpenVPN) pour accès distant sécurisé
  • Déployer un système de détection d'intrusion (Snort/Suricata)
  • Analyser les logs de sécurité et configurer des alertes

Architecture réseau sécurisée

[Schéma d'architecture sécurisée]

Internet → Pfsense (Firewall) → IDS/IPS (Snort) → Réseau interne (LAN) → Serveurs

Installation de Snort (IDS/IPS)

# Installation de Snort sur Ubuntu
sudo apt update
sudo apt install snort -y

# Configuration de Snort
sudo nano /etc/snort/snort.conf

# Définir le réseau à surveiller
ipvar HOME_NET 192.168.1.0/24

# Télécharger les règles (nécessite compte oinkmaster)
# Ou utiliser des règles communautaires

# Démarrer Snort
sudo snort -A console -q -u snort -g snort -c /etc/snort/snort.conf -i eth0

# Mode IPS (inline) nécessite une configuration plus avancée

Difficultés rencontrées

  • Problème : Règles de firewall trop restrictives bloquant le trafic légitime
  • Solution : Mise en place d'une stratégie progressive et tests approfondis
  • Problème : Configuration VPN complexe avec certificats
  • Solution : Utilisation de WireGuard (plus simple que OpenVPN)
