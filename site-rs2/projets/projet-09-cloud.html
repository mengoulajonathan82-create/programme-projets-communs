PROJET 09

Cloud Privé (Nextcloud)

Durée : 2 séances
Équipe : 3 personnes
Réalisé : Juillet 2026

Objectifs

  • Installer Nextcloud (solution de cloud privé)
  • Configurer le chiffrement côté serveur
  • Mettre en place la synchronisation multi-appareils
  • Configurer les partages sécurisés

Installation de Nextcloud

# Prérequis: LAMP (Linux, Apache, MySQL, PHP)
sudo apt-get update
sudo apt-get install apache2 mysql-server php libapache2-mod-php php-mysql php-xml php-mbstring php-zip php-gd php-curl

# Télécharger Nextcloud
cd /var/www/html
sudo wget https://download.nextcloud.com/server/releases/latest.zip
sudo unzip latest.zip
sudo chown -R www-data:www-data nextcloud
sudo chmod -R 755 nextcloud

# Base de données
sudo mysql -u root -p
CREATE DATABASE nextcloud;
CREATE USER 'nextcloud'@'localhost' IDENTIFIED BY 'mot_de_passe';
GRANT ALL PRIVILEGES ON nextcloud.* TO 'nextcloud'@'localhost';
FLUSH PRIVILEGES;
EXIT;

# Configurer Apache
sudo nano /etc/apache2/sites-available/nextcloud.conf
# Ajouter: Alias /nextcloud "/var/www/html/nextcloud/"

sudo a2ensite nextcloud
sudo systemctl reload apache2

# Accéder à http://serveur/nextcloud pour finaliser l'installation
